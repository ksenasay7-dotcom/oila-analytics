# Design tokens — OILA Analytics Platform

## Цвет
- BLACK: #0E0E0F (фон, основной)
- BLACK-SOFT: #18181B (карточки, поверхности)
- BLACK-ELEVATED: #212124 (hover/elevated поверхности)
- ORANGE: #FF6A1A (основной акцент, CTA, активные элементы)
- ORANGE-DEEP: #C9501A (hover state, обводки)
- ORANGE-DIM: #3A2415 (фоновые подсветки, бейджи)
- WHITE: #F5F3EF (основной текст, тёплый белый — не чисто белый)
- GRAY: #8A8A8E (вторичный текст)
- GRAY-LINE: #2A2A2D (разделители, границы)
- SUCCESS: #4ADE80 / DANGER: #F85149 (для роста/падения показателей)

## Типографика
- Display: 'Neue Montreal' / fallback Arial — для крупных чисел KPI и заголовков
- Body: Inter / fallback Arial — для текста и таблиц
- Mono: 'JetBrains Mono' / fallback monospace — для цифр в таблицах (выравнивание разрядов)

## Принцип
Тёмный кабинет в духе "терминала трейдера" / премиального финансового дэшборда.
Оранжевый используется ТОЧЕЧНО: главное число KPI, активный таб, кнопка загрузки,
restock/положительная динамика. Всё остальное — чёрный, оттенки серого, тонкие линии.
Никаких градиентов на фонах. Лёгкое свечение (glow) допустимо только на акцентных
элементах (активная точка на графике, кнопка).
