# OILA Analytics — платформа бизнес-аналитики

Личный кабинет для загрузки данных (выручка, расходы, ABC-анализ блюд) и
просмотра аналитики по торговым точкам Kafe Oila / G-Consulting.

## Что внутри

- `frontend/` — Next.js приложение (личный кабинет, дашборды, загрузка файлов)
- `database/schema.sql` — схема базы данных для Supabase
- `design/tokens.md` — палитра и стиль платформы

## Как развернуть (15-20 минут, бесплатно)

### Шаг 1. База данных — Supabase

1. Зайди на https://supabase.com и зарегистрируйся (можно через Google)
2. Нажми **New Project**, придумай название (например `oila-analytics`) и пароль для базы — сохрани пароль куда-нибудь
3. Подожди 1-2 минуты, пока проект создастся
4. В левом меню открой **SQL Editor** → **New query**
5. Открой файл `database/schema.sql` из этого проекта, скопируй весь текст и вставь в редактор
6. Нажми **Run** — это создаст все нужные таблицы и добавит две точки (OILA ЦЕНТР-1 и Бар Oyla)
7. В левом меню открой **Project Settings → API**
8. Скопируй два значения: **Project URL** и **anon public key** — они понадобятся на шаге 3

### Шаг 2. Создание пользователя (твой логин)

1. В Supabase, в левом меню открой **Authentication → Users**
2. Нажми **Add user → Create new user**
3. Введи свой email и пароль — это и будет вход в личный кабинет
4. Обязательно включи галочку **Auto Confirm User**

### Шаг 3. Деплой сайта — Vercel

1. Зайди на https://vercel.com и зарегистрируйся (можно через GitHub)
2. Тебе нужно будет залить папку `frontend` в свой репозиторий на GitHub:
   - Если у тебя ещё нет GitHub-аккаунта — зарегистрируйся на https://github.com
   - Создай новый репозиторий, загрузи туда содержимое папки `frontend`
3. В Vercel нажми **Add New → Project**, выбери этот репозиторий
4. На странице настроек перед деплоем найди раздел **Environment Variables** и добавь:
   - `NEXT_PUBLIC_SUPABASE_URL` = значение из шага 1.7 (Project URL)
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY` = значение из шага 1.7 (anon public key)
5. Нажми **Deploy**

Через 1-2 минуты сайт будет готов по адресу вида `oila-analytics.vercel.app`.
Заходи туда с email и паролем, которые создала в шаге 2.

## Структура Excel-файла для загрузки

Файл должен содержать листы (названия гибкие, ищутся по ключевым словам):

**Лист с операциями** (ищется по словам: операции, приход, расход, движение, дебет)
| Дата | Тип | Сумма | Способ оплаты | Статья |
|------|-----|-------|---------------|--------|
| 2026-03-01 | Приход | 1500000 | Наличные | |
| 2026-03-01 | Расход | 500000 | | Аренда |

**Лист с ABC-анализом** (ищется по словам: abc, абс, блюда)
| Блюдо | Склад | Категория | Количество | Выручка | ABC |
|-------|-------|-----------|------------|---------|-----|
| Плов | Кухня | Горячее | 45 | 2250000 | A |

Если структура твоих реальных файлов отличается — пришли пример, и я подправлю
парсер именно под него.
